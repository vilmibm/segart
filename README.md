# segart

Segmenting periodical issues into articles.

## About

`segart` is a project for taking digitized periodical issues (newspapers, magazines, journals) and segmenting them into individual articles. The goal is to turn page-level scans and OCR into article-level units suitable for search, citation, and reuse.
